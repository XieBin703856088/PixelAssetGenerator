# 更新日志 / Changelog

## [0.7.0] - 2026-07-15

这是一次面向独立游戏开发者的大版本升级，重点重构了节点图、像素算法、动画/粒子/物理系统、本地 AI 生成和整体界面。

### 新增

- 完整动画工作流：独立工作流选择、播放/暂停/停止、循环模式、时间轴、动作预设、路径和精灵特效。
- 完整粒子工作流：发射器、力场、行为、碰撞、拖尾、光照、渲染以及一体化粒子元节点。
- 物理系统：刚体模拟、约束、力场、软体网格和精灵物理动画。
- 智能纹理节点：苔藓、腐蚀、受潮、积尘、霜冻、烧灼、裂纹、崩边和调色清理。
- 本地 AI 图像节点：内置 Stable Diffusion 像素 LoRA、中文翻译模型、提示词模板、参考图控制和像素后处理。
- AMD 显卡支持：StableDiffusion.NET Vulkan 后端，同时保留 CUDA 与 CPU 回退。
- 颜色提取、缓存、常量、图像分析、输出、预览、文字等实用节点的完整实现。
- 节点画布右键菜单、复制、删除、移动、重命名、工作流模板和节点连接验证。

### 改进

- 节点库扩展至 178 个节点，覆盖 12 个类别。
- 重构树木、岩石、草地、布料、皮革、地板、石板、墙壁、水流、闪电、史莱姆、锁甲、雨水、大理石、鳞片、铁锈、电路、蜂窝、栅栏和烟雾等像素算法。
- 重构整体 UI、窗口、菜单、属性面板、布局、中文显示和滑块视觉对齐。
- 改进 32×32/64×64 像素风格约束、调色板量化、轮廓、像素簇清理与构图引导。
- 多动画/粒子工作流的状态彼此隔离，修改一个工作流不再重启其他工作流。
- 节点连接线按稳定端口键保存，并修正连接端点中心位置。
- 图求值缓存、像素缓冲池和持久状态管理得到优化。

### 修复

- 修复粒子预设覆盖用户速度、寿命、重力等参数的问题。
- 修复已有粒子不实时响应速度、寿命、力场和行为参数的问题。
- 修复普通参数拖动导致整套粒子/物理仿真反复重置的问题。
- 修复软体节点修改网格、形状、固定边和刚度后不重建，以及圆形拓扑空引用崩溃。
- 修复物理水平重力、地面摩擦和反弹参数只在初始化时生效。
- 修复音频响应通道、HSV 加权量化、智能取景像素吸附等参数未进入算法。
- 修复输出尺寸参数键错误，以及资源节点多输出端口只返回主输出的问题。
- 审计全部 C# 节点和 142 个资源节点，清除无效或仅显示但未执行的参数。

### 验证

- .NET 9 Release 编译通过：0 警告、0 错误。
- AI 参考图、动画、粒子、物理、智能纹理、参数持久化与多输出节点系统冒烟测试通过。
