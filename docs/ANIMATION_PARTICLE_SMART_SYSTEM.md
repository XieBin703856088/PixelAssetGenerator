# 动画、粒子与智能节点系统

## 1. 动画系统

动画由 `AnimationPlaybackService` 统一驱动，`PixelGraphContext` 向所有节点传递当前帧、归一化时间、帧率和累计秒数。时间轴拖动会暂停播放并重新建立目标帧的粒子状态，因此预览、逐帧导出和回退查看使用同一套确定性求值结果。

新增节点：

- **动画时间轴**：输出时间、秒数、帧进度、节拍和播放方向。
- **动作预设**：输出待机、漂浮、震动、跳跃、受击、环绕等变换通道。
- **精灵表动画**：按行列切分精灵表，支持循环、往返、单次与反向播放。

推荐链路：`动作预设 → 动画变换`，或 `动画时间轴/帧进度 → 精灵表动画`。

## 2. 粒子系统

粒子仍采用可组合的强类型链路：

`粒子发射器 → 粒子力场 → 粒子生命周期行为 → 碰撞/拖尾 → 粒子渲染`

发射器新增火焰、烟雾、雨水、雪花、火花、魔法、尘土、爆炸、气泡和落叶预设，并提供强度、尺寸、寿命随机、大小随机、二维重力/风力与预热。渲染器在“跟随预设”模式下自动选择火焰、雨滴、雪花、叶片、气泡、符文等像素纹理和适合的混合模式。

生命周期行为节点提供大小脉动、明暗闪烁、曲折运动、速度环绕、空气阻力和色相循环。行为只作用于通过粒子端口连接的上游发射器，不会污染画布中的其他粒子系统。

## 3. 智能节点

智能节点全部本地运行，不依赖网络或新增模型：

- **智能主体取景**：识别透明主体或角落背景色，自动裁空、缩放、居中或落地。
- **智能像素整理**：针对 32/64 像素限制调色板、量化透明度并清除孤立像素块。
- **智能参考配色**：提取参考图主色板并迁移到目标素材，可保留原始明暗层次。

AI 图像推荐后处理链路：

`AI 图像生成 → 智能主体取景 → 智能参考配色（可选） → 智能像素整理 → 输出`

## 4. 性能与一致性

- 所有动画和智能节点均声明求值特征，时间相关节点不会被跨帧错误缓存。
- 粒子预热最多 300 个固定步，避免异常参数阻塞 UI。
- 时间轴非顺序跳转时从第 0 帧确定性重建粒子状态，连续播放仍只计算一个新帧。
- 像素整理与配色按像素线性扫描，默认调色板最多 24 色，适合 32×32 和 64×64 素材实时预览。
